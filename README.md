# Numerical Methods in Physics Labs

This repository contains my laboratory work from the **Numerical Methods in Physics** course at **AGH University of Science and Technology**.

Each lab includes a **C++ implementation** of the numerical method and a short **report** with results, plots, and conclusions.

## Contents

- C++ source code
- numerical results
- lab reports
- plots used in reports

## Topics

- solving nonlinear equations
- matrix methods
- eigenvalue problems
- interpolation
- approximation methods
- numerical fitting
- optimization methods

## Technologies

`C++` `OpenBLAS` `LAPACK` `Numerical Methods` `Computational Physics`

## OpenBLAS / LAPACK

Some labs use **OpenBLAS** and **LAPACK** for matrix operations and solving linear algebra problems, such as systems of equations, eigenvalue problems, and matrix diagonalization.

These libraries provide efficient numerical routines which are useful in computational physics tasks.
